Hexo 安装

cnpm install -g hexo-cli

初始化博客

  1. 在已知文件夹下建立一个空文件夹–>blog

  1. 在新建文件夹（Blog）所在目录右键点击“Git Bush Here”，输入以下命令：hexo init blog
  2. 本地启动Hexo博客：hexo s（此命令用于在平时写完博客后的预览）
  3. 新建博客命令：hexo n "{filename}"
  4. 清理：hexo clean
  5. 生成：hexo g
