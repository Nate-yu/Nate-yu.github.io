使用Next主题优化博客

发表于 2020-07-08 | 分类于 Hexo

下载Next主题

在博客根目录下打开Git并输入以下命令：git clone https://github.com/iissnan/hexo-theme-next themes/next

增加标签页和分类页

修改next文件中的_config.yml文件，在menu字段中启用tags和categories

打开标签页与分类页

  1. 在blog文件下打开Git输入命令：hexo n page tags/categories
  2. 修改各自的index.md文件：加入字段“type”，设置为tags/categories
  3. 注意：如果出现访问失败，并且url里面莫名多了一个%20，则将空格都删掉即可

主题样式的更改

在next文件夹下修改_config.yml文件中的Schemes字段即可

更换头像

在next文件夹下修改_config.yml文件中的avatar字段即可

启用社交账号链接

在next文件夹下修改_config.yml文件中的social字段即可

启用友情链接

在next文件夹下修改_config.yml文件中的links字段即可

Build Blog

发表于 2020-07-07

部署环境

Node.js下载

Node.js官网下载

Node.js网盘下载 提取码：rgh8（永久有效）

Git下载

Git官网下载

Git网盘下载 提取码：3ep7（永久有效）

利用npm安装cnpm的镜像源

npm install -g cnpm --registry=https://registry.npm.taobao.org

Hexo 安装

cnpm install -g hexo-cli

初始化博客

  1. 在已知文件夹下建立一个空文件夹–>blog
  2. 在新建文件夹（Blog）所在目录右键点击“Git Bush Here”，输入以下命令：hexo init blog
  3. 本地启动Hexo博客：hexo s（此命令用于在平时写完博客后的预览）

写博客

  1. 新建博客命令：hexo n "{filename}"
  2. 清理：hexo clean
  3. 生成：hexo g

部署到远端

  1. 在个人的GitHub新建一个以个人用户名为名字的仓库，比如我的：Nate-yu.github.io
  2. 在新建文件夹（blog）下安装git插件：cnpm install --save hexo-deployer-git
  3. 设置”_config.yml”文件
  4. 部署到远端：hexo d

更换主题

  1. 从github上clone主题到themes目录：git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
  2. 修改”_config.yml”文件（将主题修改为将要使用的主题）
  3. 清理，生成，部署到本地，部署到远端：hexo clean –> hexo g –> hexo s –> hexo d
